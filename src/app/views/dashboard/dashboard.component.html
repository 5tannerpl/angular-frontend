This is a small function model in a Child and Family Health Nursing Management App.

One of the task is for the registered nurse in Child and Family Health Center to visit moms and newborn baby,which is a free NSW goverment service.

Nurses will rate the healthy status each time and book the day for the next visit.

In this funtiaonal widgets, the number of overdue visits and the number of toltal contacted moms are displayed as an item in the dashboard when nurse leader login. More details will be shown up in table while clicking the widget.
